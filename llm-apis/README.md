## Getting Started

### Add .env file

Add .env file with following content:

```bash
OPENAI_API_KEY=<key>
```

Then, install dependencies:

```bash
npm i
```

Last but not least, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
